# AgentCoin Assets

This directory contains official AgentCoin visual identity assets and metadata.

## Images

### Logo (`images/agentcoin-logo.png`)
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Usage**: Profile pictures, app icons, circular displays
- **Theme**: Blue gradient with AI agent silhouette and AGNT typography

### Banner (`images/agentcoin-banner.png`) 
- **Size**: 1792x1024px (optimized for Twitter headers)
- **Format**: PNG
- **Usage**: Social media headers, website banners
- **Theme**: Human-AI collaboration with "AgentCoin - Economy for AI Agents" text

## Metadata

### `metadata.json`
Standard token metadata file containing:
- Token information (name, symbol, decimals)
- Contract addresses and network details
- Visual asset URLs (logo, banner)
- Social media links
- Current supply and partnership information

**Usage**: 
- Wallet integrations
- Exchange listings
- DeFi protocol interfaces
- Block explorer displays
